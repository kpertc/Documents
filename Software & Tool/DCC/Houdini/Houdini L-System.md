`F` Forward
`f` Forward no geometry

`+` Right
`-` Left

`!` Thickness Scale
`"` Step Size Scale

`~` Random amount (Pitch / Roll / Turn)

`:` Probability

`&` Pitch Up
`^` Pitch Down

`\\` Row CW
`/` Row CCW

`[]` Branch
`()` Explicit value
